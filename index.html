Css

Css (Cascade Style Sheets) it's the "second part" of our web pages, the language that we are gonna use to create the style or "design" of our page using the Html that we have like our base.

JavaScript

JavaScript it's the last part of our "Web trinity", the actual programming language that we can use to make our web page "interactive".
